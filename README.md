# pyghlight -- a simple wrapper for pygments to generate a pdf file

## Requirements

lualatex needs to be installed and placed in your command path.

## Install

```
pip install .
```

or 

```
pip install -e . # editablel mode
```


## How to use

    pyghlight sample.py
